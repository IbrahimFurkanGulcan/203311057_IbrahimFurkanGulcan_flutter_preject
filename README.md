## 📦 Kullanılan Temel Paketler
Projeyi geliştirirken aşağıdaki güncel kütüphaneler kullanılmıştır:
* `firebase_core`: Firebase ana bağlantısı
* `cloud_firestore`: Gerçek zamanlı NoSQL veritabanı işlemleri (StreamSubscription ile anlık UI güncellemeleri)
* `firebase_auth`: Kullanıcı kayıt, giriş ve yetkilendirme (Role-based) işlemleri
* `flutter/services`: Form doğrulama (FilteringTextInputFormatter, Regex) işlemleri
